## About Gravity

Gravity is a production-ready Node.js SaaS boilerplate created by Kyle Gawley.

### Key Features

- **Subscription Billing**: Stripe integration with trials, free plans, seat/usage billing
- **Authentication**: Email/password, magic links, 500+ social networks, 2FA
- **UI Templates**: 50+ components, 25+ views, dark mode, built with Shadcn
- **Database Support**: MySQL, MongoDB, PostgreSQL, MariaDB
- **REST API**: Token auth, protected endpoints, API keys, throttling
- **Multi-tenant**: Organizations, invitations, user roles
- **Mobile Apps**: Native iOS & Android with React Native
- **Admin Dashboard**: Metrics, user management, event logs (Mission Control)
- **Email System**: 20+ multilingual templates, 20+ email services
- **AI Support**: Cursor & Windsurf rules included
- **Localization**: Multi-language support
- **Testing**: 40+ integration tests
- **Landing Page**: SEO-optimized Astro template

### Technology Stack

- **Backend**: Node.js
- **Frontend**: React
- **Mobile**: React Native
- **Language**: JavaScript (TypeScript available)
- **Databases**: MySQL, MongoDB, PostgreSQL, MariaDB
- **Payments**: Stripe
- **UI**: Shadcn components
